CSS Animation on Scroll

Animate elements as the user scrolls down and the section is visible. This is a easy to apply libary with minimum setup.

Library is Free and Open Source.

HOW TO USE LIVE DEMO
How to Use

  1. Add animate.css in your application.


    CDN 

    Add the following on your page.

    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/3.7.0/animate.min.css" /> https://cdnjs.com/libraries/animate.css/

    Or alternatively you can download the library and link on your project.

    OFFICIAL WEBSITE (Download animate.css from here)
    https://daneden.github.io/animate.css/

  2. Add OyeThemes Scroll Animation JS PLugin

    Download Library to your JS Folder.

    https://github.com/shishirraven/animate-on-scroll/blob/master/oyethemes_onscroll_animation.js

    Include in your project as follows

    <script src="path-to-your-js/oyethemes_onscroll_animation.js"></script>

  3. Add Animation to Element.

    On the Selected element add the following classes.

    a. animated
    b. scrollanimation
    c. Animation your choose from animate.css for example. "bounce", "flip"...

    Once you apply all these the element should look like.

    <div class="animated scrollanimation bounce">...</div>

  4. Done
